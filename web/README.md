# DanceProgressTracker — Web (front-end)

React + Vite. Front-end-частина системи: окремий інтерфейс для **викладача** і
для **адміністратора** (за ролями), з підтримкою інтернаціоналізації (uk/en).

## Запуск

```bash
npm install
npm run dev          # http://localhost:5173
```

Демо-доступи: `admin/admin` (адміністратор), `teacher/teacher` (викладач).

## Режими роботи

- **Демо (за замовчуванням):** дані зберігаються у `localStorage`, бекенд не потрібен.
- **З реальним API:** задати `VITE_API_URL=http://localhost:4000` — застосунок
  звертається до Express-бекенда (`../backend`).

## Інтернаціоналізація / локалізація (i18n / l10n)

- Перемикання української та англійської мов «на льоту» (`i18next`), вибір зберігається.
- Формат дати за регіоном через `Intl.DateTimeFormat` (`uk-UA` / `en-US`).
- Порядок сортування ПІБ через `Intl.Collator` за активною мовою.
- Напрям тексту виставляється атрибутом `document.dir` із конфігурації локалі (підтримка RTL).

## Адміністрування

- Управління користувачами (ролі admin/teacher).
- Управління даними (групи, учні).
- Резервна копія даних (експорт JSON) та відновлення (імпорт JSON).
- Експорт зведеного звіту у CSV.

## Структура

```
src/
  api.js            доступ до даних (реальний REST або демо на localStorage)
  auth.jsx          контекст автентифікації + ролі
  i18n/             конфіг локалей, переклади uk/en, формат дати, сортування
  components/       Layout (шапка, навігація, перемикач мови), Modal
  pages/            Login, Groups, GroupDetail, Users, DataManagement
  App.jsx           маршрути + захист за роллю
```
